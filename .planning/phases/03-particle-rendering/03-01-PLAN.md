  <name>Task 1: Create Camera class with orbit controls</name>
  <files>src/Camera.cs</files>
  <action>Create a new Camera.cs file implementing an orbit camera using spherical coordinates.

  **Camera structure:**
  - Position stored as spherical coordinates: distance, yaw (azimuth), pitch (elevation)
  - Target point (center of orbit) - default to origin (0,0,0)
  - Up vector - default to (0,1,0)
  - Projection matrix cache with configurable FOV

  **Public API:**
  ```csharp
  public class Camera
  {
      public Camera(float width, float height);
      public void UpdateOrbit(float deltaYaw, float deltaPitch);  // Left-drag
      public void UpdateZoom(float deltaZoom);                     // Scroll wheel
      public void UpdatePan(float deltaX, float deltaY);           // Right-drag
      public Matrix4 GetViewMatrix();
      public Matrix4 GetProjectionMatrix();
      public void Resize(float width, float height);
  }
  ```

  **Spherical to Cartesian conversion:**
  ```
  x = distance * cos(pitch) * sin(yaw)
  y = distance * sin(pitch)
  z = distance * cos(pitch) * cos(yaw)
  ```

  **Constraints from PITFALLS:**
  - Clamp pitch to [-89, 89] degrees to prevent gimbal lock
  - Clamp distance to [1, 50] range for sensible zoom limits
  - Pan modifies target point, not camera position directly

  **Implementation details:**
  - Use OpenTK.Mathematics.Matrix4, Vector3
  - Matrix4.LookAt for view matrix
  - Matrix4.CreatePerspectiveFieldOfView for projection (45 deg, 0.1 near, 100 far)
</action>

     ```csharp
         particleCount * ParticleData.SizeInBytes,
         IntPtr.Zero,  // Orphan existing buffer

         IntPtr.Zero,
         particleCount * ParticleData.SizeInBytes,
         particles);
     ```

  5. **RenderParticles** (instanced draw call):
     ```csharp
     GL.UseProgram(_particleShaderProgram);

     // Bind SSBO to binding point 0
     GL.BindBufferBase(BufferRangeTarget.ShaderStorageBuffer, 0, _particleSSBO);

     // Set uniforms: mvp matrix, particle size
     Matrix4 mvp = _camera.GetViewMatrix() * _camera.GetProjectionMatrix();
     GL.UniformMatrix4(_mvpUniformLocation, false, ref mvp);
     GL.Uniform1(_particleSizeUniformLocation, _particleSize);

     // Instanced draw: 4 vertices, _particleCount instances
     GL.BindVertexArray(_quadVAO);
     GL.DrawArraysInstanced(PrimitiveType.TriangleStrip, 0, 4, _particleCount);
     ```

     ```csharp
     if (_particleSSBO != 0) GL.DeleteBuffer(_particleSSBO);
     if (_quadVBO != 0) GL.DeleteBuffer(_quadVBO);
     if (_quadVAO != 0) GL.DeleteVertexArray(_quadVAO);
     ```

  **particle.vert changes:**
  ```glsl
  #version 430 core  // Need 430 for SSBO

  layout(location = 0) in vec2 aVertex;  // Base quad vertex

  // SSBO with particle data
  layout(std430, binding = 0) buffer ParticleData {
      vec4 data[];  // Position.xyz in data[i*2], Color in data[i*2+1]
  } particles;

  uniform mat4 mvp;
  uniform float uPointSize;

  out vec4 vColor;

  void main() {
      uint idx = uint(gl_InstanceID) * 2u;  // Each particle = 2 vec4s
      vec3 position = particles.data[idx].xyz;
      vColor = particles.data[idx + 1u];

      // Expand vertex by particle size
      vec2 offset = aVertex * uPointSize * 0.01;  // Scale factor

      // Apply MVP transform
      gl_Position = mvp * vec4(position + vec3(offset, 0.0), 1.0);
  }
  ```

  Implement handlers:
  ```csharp
  private void OnMouseDown(object? sender, MouseEventArgs e) {
      if (e.Button == MouseButtons.Left) {
          _isLeftDragging = true;
          _lastMousePos = e.Location;
      } else if (e.Button == MouseButtons.Right) {
          _isRightDragging = true;
          _lastMousePos = e.Location;
      }
  }

  private void OnMouseUp(object? sender, MouseEventArgs e) {
      if (e.Button == MouseButtons.Left) _isLeftDragging = false;
      if (e.Button == MouseButtons.Right) _isRightDragging = false;
  }

  private void OnMouseMove(object? sender, MouseEventArgs e) {
      if (_isLeftDragging) {
          float deltaX = (e.X - _lastMousePos.X) * OrbitSensitivity;
          float deltaY = (e.Y - _lastMousePos.Y) * OrbitSensitivity;
          _glHost?.OrbitCamera(deltaX, deltaY);
          _lastMousePos = e.Location;
      } else if (_isRightDragging) {
          float deltaX = (e.X - _lastMousePos.X) * PanSensitivity;
          float deltaY = (e.Y - _lastMousePos.Y) * PanSensitivity;
          _glHost?.PanCamera(deltaX, -deltaY);  // Invert Y for natural feel
          _lastMousePos = e.Location;
      }
  }

  private void OnMouseWheel(object? sender, MouseEventArgs e) {
      float delta = e.Delta * ZoomSensitivity / 120f;  // Delta is in 120 increments
      _glHost?.ZoomCamera(delta);
  }
  ```

  **Cleanup in OnFormClosing:**
  The event handlers will be cleaned up automatically with the GLControl disposal.
